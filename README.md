# Лекции по основам алгоритмизации и программирования

Курс посвященный основам алгоритмизации и программирования на языке Python. Материал охватывает базовые понятия алгоритмов, структуры данных, управляющие конструкции и основы написания программ.

## Лекции

1. [Основы алгоритмизации](lectures/лекция_1_алгоритм_и_блок_схема.md) — понятие алгоритма, его свойства (дискретность, детерминированность, результативность, конечность, массовость), способы описания алгоритмов, блок-схемы по ГОСТ.

2. [Циклы и сортировка массивов](lectures/лекция_2_циклы_и_сортировка_массивов.md) — принцип работы циклов, псевдокод, понятие массива, алгоритмы сортировки (пузырьковая, выбором).

3. [Основы Python](lectures/лекция_3_основы_python.md) — введение в Python, первая программа, переменные, типы данных, ввод/вывод, условные конструкции.

4. [Функции в Python](lectures/лекция_4_функции_в_python.md) — определение и вызов функций, параметры и аргументы, возвращаемые значения, повторное использование кода.
