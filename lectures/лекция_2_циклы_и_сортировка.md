# Лекция 2: Циклы и сортировка массивов

## 1. Что такое цикл

Цикл — это способ выполнить один и тот же набор действий несколько раз.

Проще говоря: если вам нужно повторить одно и то же действие много раз, вместо копирования кода вы используете цикл.

### Пример из жизни

Представьте, что вам нужно 5 раз сказать «Привет»:

Без цикла:
- Привет
- Привет
- Привет
- Привет
- Привет

С циклом:
- «Повтори 5 раз: скажи Привет»

Цикл делает именно это — повторяет действия.

---

## 2. Как работает цикл (простое объяснение)

Цикл обычно состоит из:

1. **Начального значения** (с чего начинаем)
2. **Условия** (когда остановиться)
3. **Шага** (как изменяется значение)

### Псевдокод цикла

```
для i от 0 до 4:
    вывести "Привет"
```

### Объяснение словами

- Сначала i = 0
- Проверяем: i ≤ 4 → да
- Выполняем действие (печатаем «Привет»)
- Увеличиваем i на 1
- Повторяем, пока условие выполняется

---

## 3. Массив и зачем его сортировать

**Массив** — это список значений.

Пример:

```
[5, 2, 4, 1]
```

**Сортировка** — это упорядочивание элементов, например:

```
[1, 2, 4, 5]
```

---

## 4. Одна итерация сортировки (простой шаг)

Давайте посмотрим, что происходит за **один проход по массиву**.

### Идея

Мы сравниваем соседние элементы и меняем их местами, если они стоят неправильно.

### Псевдокод одной итерации

```
массив = [5, 2, 4, 1]

для i от 0 до длина(массив) - 2:
    если массив[i] > массив[i + 1]:
        поменять местами массив[i] и массив[i + 1]
```

### Объяснение словами

Берём массив: `[5, 2, 4, 1]`

1. Сравниваем 5 и 2 → 5 больше → меняем → `[2, 5, 4, 1]`
2. Сравниваем 5 и 4 → 5 больше → меняем → `[2, 4, 5, 1]`
3. Сравниваем 5 и 1 → 5 больше → меняем → `[2, 4, 1, 5]`

После одной итерации:

```
[2, 4, 1, 5]
```

👉 Самый большой элемент «всплыл» в конец.

---

## 5. Полная сортировка (двойной цикл)

Одной итерации недостаточно, потому что массив всё ещё не отсортирован.

Нужно повторить процесс несколько раз.

### Псевдокод полной сортировки (пузырьковая сортировка)

```
массив = [5, 2, 4, 1]

для j от 0 до длина(массив) - 1:
    для i от 0 до длина(массив) - 2:
        если массив[i] > массив[i + 1]:
            поменять местами массив[i] и массив[i + 1]
```

---

## 6. Объяснение двойного цикла словами

### Внешний цикл (j)

Говорит: «Повтори проход по массиву несколько раз»

### Внутренний цикл (i)

Проходит по массиву и делает сравнения.

---

### Как это работает шаг за шагом

Исходный массив:

```
[5, 2, 4, 1]
```

#### Первый проход:

→ `[2, 4, 1, 5]`

#### Второй проход:

→ `[2, 1, 4, 5]`

#### Третий проход:

→ `[1, 2, 4, 5]`

Теперь массив отсортирован.

---

## 7. Простое объяснение идеи

Можно представить, что:

- мы снова и снова «проходимся» по списку
- каждый раз «всплывает» самый большой элемент
- через несколько проходов всё оказывается на своих местах

---

## 8. Итог

- Цикл позволяет повторять действия
- Один цикл — один проход
- Два цикла — повторение проходов
- Сортировка — это многократное сравнение и обмен элементов
