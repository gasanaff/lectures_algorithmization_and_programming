# Лекция 3: Основы Python

## 1. Что такое Python

Python — это язык программирования, который легко читать и писать.

Его используют для:
- создания сайтов
- анализа данных
- автоматизации задач
- написания скриптов

Главное преимущество — простой и понятный синтаксис.

---

## 2. Первая программа

```
print("Привет, мир!")
```

### Объяснение

- `print` — команда вывести текст на экран
- всё, что в кавычках — это текст

---

## 3. Переменные

Переменная — это «коробка» для хранения данных.

### Пример

```
x = 5
name = "Аня"
```

### Объяснение

- `x` хранит число 5
- `name` хранит текст "Аня"

---

## 4. Типы данных

Основные типы:

- Числа:
```
x = 10
y = 3.14
```

- Строки (текст):
```
text = "Привет"
```

- Логические значения:
```
is_ok = True
```

---

## 5. Ввод данных

```
name = input("Введите имя: ")
print("Привет,", name)
```

### Объяснение

- `input` ждёт ввод пользователя
- введённое значение сохраняется в переменную

---

## 6. Условие (if)

Позволяет выполнять код только при выполнении условия.

### Пример

```
age = 18

if age >= 18:
    print("Можно заходить")
```

### Объяснение

- если возраст больше или равен 18 → выполняется код
- отступы (пробелы) важны!

---

## 7. Цикл в Python

### Цикл for

```
for i in range(5):
    print(i)
```

### Объяснение

- `range(5)` создаёт числа от 0 до 4
- цикл проходит по каждому числу

---

## 8. Списки (массивы)

```
numbers = [1, 2, 3, 4]
```

### Доступ к элементам

```
print(numbers[0])  # 1
```

---

## 9. Цикл по списку

```
numbers = [1, 2, 3]

for num in numbers:
    print(num)
```

### Объяснение

- `num` по очереди принимает значения из списка

---

## 10. Простой пример: сумма чисел

```
numbers = [1, 2, 3, 4]
sum = 0

for num in numbers:
    sum = sum + num

print(sum)
```

### Объяснение

- создаём переменную `sum`
- по очереди добавляем каждое число

---

## 11. Итог

Вы узнали:

- как выводить данные (`print`)
- как хранить данные (переменные)
- типы данных
- условия (`if`)
- циклы (`for`)
- списки
