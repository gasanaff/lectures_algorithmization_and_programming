# Лекция 4: Функции в Python

## 1. Что такое функция

Функция — это именованный блок кода, который выполняет определённую задачу.

Проще говоря: вы один раз пишете код, а потом можете использовать его много раз.

---

## 2. Зачем нужны функции

Функции помогают:

- не повторять один и тот же код
- делать программу понятнее
- разбивать сложные задачи на простые

---

## 3. Простейшая функция

```
def say_hello():
    print("Привет!")
```

### Объяснение

- `def` — означает «создать функцию»
- `say_hello` — имя функции
- `()` — скобки для параметров

### Вызов функции

```
say_hello()
```

👉 Только после вызова функция выполнится

---

## 4. Функция с параметрами

```
def greet(name):
    print("Привет,", name)
```

### Вызов

```
greet("Аня")
```

### Объяснение

- `name` — это параметр
- при вызове мы передаём значение

---

## 5. Возврат значения (return)

Функция может не только что-то делать, но и возвращать результат.

```
def add(a, b):
    return a + b
```

### Использование

```
result = add(2, 3)
print(result)
```

### Объяснение

- `return` возвращает значение
- результат можно сохранить в переменную

---

## 6. Почему return важен

Без `return` функция просто выполняет действие.

С `return` — она отдаёт результат, который можно использовать дальше.

---

## 7. Пример: функция для суммы списка

```
def sum_list(numbers):
    total = 0
    for num in numbers:
        total = total + num
    return total
```

### Объяснение словами

- создаём переменную total
- проходим по каждому числу
- добавляем его к total
- возвращаем итог

---

## 8. Разделение программы на функции

Вместо одного большого кода:

```
# всё в одном месте
```

Лучше так:

```
def input_data():
    ...

def process_data():
    ...

def output_data():
    ...
```

👉 Код становится понятнее и удобнее

---

## 9. Частые ошибки

1. Забыли вызвать функцию
2. Забыли return
3. Неправильно передали параметры

---

## 10. Итог

Вы узнали:

- что такое функция
- как её создавать (`def`)
- как передавать параметры
- как возвращать результат (`return`)
